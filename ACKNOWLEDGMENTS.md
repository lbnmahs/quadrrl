## Third-Party Libraries and Frameworks

Quadrrl builds upon and integrates with several excellent open-source projects:

- **[NVIDIA Isaac Lab](https://isaac-sim.github.io/IsaacLab/)** - The simulation framework that powers Quadrrl's environments
- **[RSL-RL](https://github.com/leggedrobotics/rsl_rl)** - Reinforcement learning library for robotics
- **[RL Games](https://github.com/Denys88/rl_games)** - NVIDIA's reinforcement learning framework
- **[SKRL](https://github.com/Toni-SM/skrl)** - Scikit-learn compatible reinforcement learning library
- **[HARL](https://github.com/PKU-MARL/HARL)** - Multi-agent reinforcement learning framework (customized for Isaac Lab)

## Robot Models

Quadrrl uses robot models and configurations from:

- **ANYbotics** - ANYmal-C and ANYmal-D robot models
- **Boston Dynamics** - Spot robot model
- **Unitree** - Unitree Go2 robot model
